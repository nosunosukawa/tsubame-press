# TSUBAME PRESS & CO.

Starbucks Japan（starbucks.co.jp）のサイト構造・UIパターン（メガメニュー、カード型
カルーセル、編集コンテンツハブ、店舗検索）を参考に制作した、燕三条の金属加工産業
（プレス加工・鏡面研磨）を背景にした架空の食器メーカー「TSUBAME PRESS & CO.」の
コーポレート/ECサイト（受託制作実績のデモ）。実案件はなく、実在企業（本物の燕三条
メーカー・問屋・Starbucks等）とは無関係の完全架空クライアント。

## ブランド設定

1958年創業、燕三条で金型・プレス加工の下請けを営んできた金属加工会社が、3代目で
プレス製品を自社ブランドとして直販するD2Cへ転換——という設定。「プレスが、かたちを
つくる。研磨が、心を宿す。」を軸に、実在する燕三条の産業史（和釘→洋釘輸入によるプレス機
導入→金型技術の定着→金属洋食器の量産）を下敷きにしている。

カスタムオーダーは、プレス加工が金型に依存する量産方式である点を踏まえ、「型を変えずに
承れるオーダー（名前入れ・仕上げ選択・IH対応の底面プレート圧着・持ち手の素材選択・刃付け
方向）」と「金型が必要なオーダー（サイズ変更等、まとまった数量からのB2B相談）」を明確に
区別して設計した。

## サイト構成(21ページ)

- トップページ: ヒーローカルーセル、メガメニュー、新着カード群、オンラインストア商品
  グリッド、ものづくりストーリー予告、店舗CTA
- 商品カテゴリ一覧×5: カトラリー / 包丁・キッチンナイフ / 鍋・調理器具 / 食器・うつわ /
  タンブラー・ドリンクウェア
- 商品詳細×3: 鏡面ディナースプーン&フォーク / 鏡面三徳包丁 / 鏡面二重タンブラー
- ギフト特集、カスタムオーダー
- 直売所・取扱店(Google Maps埋め込み)
- ストーリー一覧 + 職人インタビュー記事×3
- 会社案内、採用情報、よくあるご質問、サイトマップ、利用規約・プライバシーポリシー

## 技術スタック

- 素のHTML / CSS / JavaScript(ビルドツールなし)
- Google Fonts(Barlow Condensed + Zen Kaku Gothic New + Noto Sans JP)
- メガメニュー・モバイルハンバーガーメニュー・ヒーローカルーセル・カードカルーセルは
  自前のvanilla JS(`script.js`)
- ネイティブ `<details>` によるFAQアコーディオン(JS不要)
- 店舗ページはGoogle Maps埋め込み(APIキー不要のiframe形式)
- 商品・職人・工場写真はすべて`gemini-image`スキルによるオリジナル生成画像。実在
  ブランドのロゴ・製品シルエットに似ないよう明示的に回避し、生成後は必ず人間目視で確認

## ディレクトリ構成

```
tsubame-press/
├── index.html
├── products-*.html        # 商品カテゴリ一覧×5
├── product-detail-*.html  # 商品詳細×3
├── gift.html / custom-order.html
├── stores.html
├── stories.html / story-*.html
├── company.html / recruit.html / faq.html / sitemap.html / policy.html
├── styles.css
├── script.js
└── images/
    ├── *.jpg   # サイトで使用する最適化済み画像
    └── raw/    # 生成直後のマスター(gitignore対象)
```

## ローカルで動かす

```sh
python3 -m http.server 8091
# http://localhost:8091/index.html
```
